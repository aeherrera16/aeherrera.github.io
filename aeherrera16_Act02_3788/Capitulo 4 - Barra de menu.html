Barra de navegación - PARTE I

Barra de navegación para patallas de moviles

-Ahora crearemos una barra de navegación para pantallas de moviles, que contendrá el enlace de la marca y un botón que al darle clic espanda los elementos que se encuentra como enlace. Para esto dentro de una etiqueta "nav" con la clase "navbar navbar-defaul" crearemos un contenedor de la clase "container-fliud", que es el contenedor que contendrá a los elementos del navegador y dentro de este crearemos un botón que solo será visualizado en pantallas pequeñas. Colocamos la etiqueta "botton" con la clase "navbar-toggle" y los atributos data-toggle="collapse" y data-target="#bs-example-navbar-collapse-1", y por último el enlace de la marca con la clase "navbar-brand".
NOTA: Para que el botón tenga la apariencia de un menú desplegable le pondremos 3 barritas, las cuales se visualizan colocando la etiqueta "span" con la clase "icon-bar".

Barra de navegación fija - PARTE VI

-Bootstrap también nos proporciona una clase llamada "navbar-fixed-top" que se coloca en la etiqueta principal "nav" y se asegura que nuestra barra se posicione en la parte superior del navegador todo el tiempo. Si te fijaste la barra de navegación que declaramos en la última parte de nuestro archivo .html es la que se visualiza en la parte superior de esta página. Esto quiere decir que con esta clase la barra de navegación se visualizará ahi independientemente de donde se declare. Pero para buenas prácticas de desarrollo web se recomiendo que la barra de navegación siempre se declare al principio.
NOTA: Al colocar nuestra barra de navegación fija en la parte superior, esta tapará parte del contenido de nuestra página, por lo que een nuestra hoja de estilo css colocaremos a nuestro "body" un "padding-top: 50px" y así podemos visualizar nuestra página con normalidad.
